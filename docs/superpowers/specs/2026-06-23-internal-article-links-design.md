# Внутренние гиперссылки на статьи — дизайн (2026-06-23)

Классическая возможность СПС (Гарант/Консультант гиперлинкуют каждую ссылку «ст. N» в
тексте): в тексте статьи упоминания «статья N» / «ст. N» делаем кликабельными ссылками на
эту же статью в текущем акте. ТК РФ обильно ссылается на свои статьи («в соответствии со
статьёй 72 настоящего Кодекса») — резко улучшает навигацию. Независимо от трека AI.

## Подход (внутри-документные, высокоточные)

Линкуем ТОЛЬКО ссылки на статьи, которые ЕСТЬ в текущей редакции этого же акта (по набору
anchors). Не пытаемся резолвить межактовые ссылки («ст. 5 ФЗ-152») — это отдельная задача;
здесь высокая точность важнее полноты.

### `documents/refs.py` — чистая функция (без Django-моделей)
- `linkify_internal_refs(text, anchors) -> SafeString`:
  1. Экранирует текст (XSS-безопасно);
  2. Разбивает на абзацы по пустым строкам, одиночные `\n`→`<br>` (как Django `linebreaks`);
  3. Находит `\bстать[а-яё]+|\bст\.?` + число `N` (вкл. дробные `312.1`); если
     `st-<slugify(N)>` ∈ `anchors` — оборачивает совпадение в `<a href="#st-N">`;
  4. Возвращает `mark_safe`. Пустые anchors → просто экранирование+абзацы (как раньше).
- Якорь строится как в `Article.save()`: `f"st-{slugify(num.replace('.','-'))}"`.

### Template-tag `documents/templatetags/doc_refs.py`
- Фильтр `linkify_refs(text, anchors)` → `linkify_internal_refs`.

### Интеграция
- `documents/views.py:document_detail` — добавляет в контекст `anchors = {a.anchor for a in
  articles if a.anchor}` (статьи уже загружены — без доп. SQL).
- `_article_node.html` — `{% load doc_refs %}`; строку `{{ node.text|linebreaks }}` заменить
  на `{{ node.text|linkify_refs:anchors }}` (фильтр сам делает абзацы). `anchors` доступен
  через наследуемый контекст include (line 53 без `only`).

## Тестирование
`documents/tests/test_refs.py` (pure, без БД): линкует «статьёй 72» когда st-72 в наборе;
НЕ линкует когда нет; экранирует HTML (`<b>`→`&lt;b&gt;`, не тег); дробный номер 312.1→
st-312-1; абзацы/переносы сохраняются; не линкует «и 81» без префикса (высокая точность).
`documents/tests/test_internal_links_view.py` (django_db): reader-страница ТК с двумя
статьями, где ст.81 ссылается на ст.72 → HTML содержит `href="#st-72"`. Hotspot test_views.py
не трогаем.

## Риск
Аддитивно: новый модуль+тег, правка одной строки reader-шаблона и контекста view. XSS-безопасно
(экранируем до линковки). Ноль моделей/миграций. Обратимо.
